# Shell
The shell is a program that takes keyboard commands and passes them to the OS
Almost all linux distros use bash from the GNU project
use bash command to start bash session on mac instead of zsh
[me@linuxbox ~]$ - is the shell prompt
If the last character has # instead of $ the user has super user privileges 
using the up arrow shows us the command history
have to use shift-ctrl-c and v to use them on the terminal
date - display current time and date 
uptime - how long system has been running
virtual terminals can be accessed using ALT-F#
df - current amount of free space on disk drives
free - current free memory
exit or ctrl-D to end session

# Navigation
Uses a hierarchical directory structure
This means it is a tree like pattern of directories or folders 
The first directory is called the root directory
parent directory - directory above current
current - where you are current located
pwd - displays current working directory
Each user is given a home directory and it is the only place a regular user is allowed to write files
ls - list files and directories in current directory 
cd - change directory include path of destination after cd 
a pathname is route we take to get to a directory 
absolute pathname - begins with the root down to directory
relative pathname - starts from current directory down to target
. - current directory
.. - parent directory
ls -a to list all hidden files 
if a file start with a . it is hidden
cd alone will bring you back to home directory
cd .. back one directory
cd ~user will change to user directory
